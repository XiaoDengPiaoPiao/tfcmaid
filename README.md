### 🔄 **隙间系统详解**

**坐标IO设置**：
```
[界面布局]
┌──────────────┐
│   女仆       │ ←上面
├──────────────┤
│   箱子        │ ←下面
└──────────────┘
↑箭头方向：表示物品流向
```

**黑白名单机制**：
- **白名单模式**：只允许列表中的物品传输
- **黑名单模式**：禁止列表中的物品传输
- **隙间状态检查**：如出现异常交互，请检查：
  1. 隙间是否已绑定箱子
  2. 当前是黑名单还是白名单模式
  3. 您的物品是否在对应名单中

**特殊功能说明**：
1. **挤奶功能**：自动从绑定的隙间搜索空容器
2. **繁殖功能**：遍历隙间或女仆背包寻找饲料
3. **屠夫功能**：掉落物优先存入隙间，满后无视
